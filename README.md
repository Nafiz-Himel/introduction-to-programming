This repo helps you (and me) understand basic programming building blocks such as:

- Data types and variables  
- Loops and nested loops  
- Arrays and 2D arrays  
- Strings and string operations  
- Functions  
- Pointers and memory concepts  
- Recursion  

These topics form the foundation for writing programs, solving problems, and understanding more advanced CS concepts.
